# Product Guidelines: muaz

## Visual Identity
- **Design Style:** Classic and Elegant (Mobile-optimized)
- **Primary Palette:** Deep Navy and Cream
- **Dark Mode Support:** A built-in dark mode option for a professional experience.
- **Typography:**
  - **Headings:** Serif font (e.g., Playfair Display) for a sophisticated look.
  - **Body Text:** Clean serif font (e.g., Lora) for readability.
- **Interactive Elements:**
  - **Order Button:** Prominent, styled to complement the classic aesthetic.
  - **Billing Popup:** Clean, intuitive, and mobile-friendly.

## Prose & Messaging
- **Tone:** Professional, persuasive, and informative.
- **Goal:** To highlight the value of the book and guide users seamlessly through the purchase process.
- **Telegram Confirmation:** User data is securely captured and sent to the seller for order fulfillment.
