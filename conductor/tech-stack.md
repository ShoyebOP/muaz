# Tech Stack: muaz

## Frontend
- **Framework:** React (TypeScript)
- **Routing:** React Router for multi-page navigation.
- **Styling:** Vanilla CSS (Tailored for classic, elegant design)
- **Responsiveness:** Mobile-first approach for optimization across devices.

## Backend
- **Integration:** Vercel Serverless Functions
- **Messaging API:** Telegram API for capturing and sending customer details.
- **Hosting:** Vercel (Integrated via GitHub)

## Tools & Libraries
- **Version Control:** Git & GitHub
- **Package Manager:** NPM
- **Testing:** Vitest and React Testing Library for automated unit and integration tests.
